# Blih CLI
This project is the Blih (Bocal Lightweight Interface for Humans) Js CLI (Command-Line Interface) for linux env.

### Setup
1. Install node & npm https://nodejs.org/en/
3. `git clone https://github.com/GreenDjango/blih_cli`
4. `cd blih_cli/`
5. `npm install`
6. `npm run build`
7. `./index.js`

### Lint
```
npm run lint
```
### Build
```
npm run build
```
